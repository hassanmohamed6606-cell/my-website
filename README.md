Shukri Bakery

A simple, responsive bakery website built to demonstrate modern CSS layout techniques.

 Features
- 3 Pages: Home, About, and Contact.
- Flexbox & Grid: Used CSS Grid for the product gallery on the homepage and Flexbox for the navigation bar and about page layout.
- No Frameworks: Pure HTML5 and CSS3.
- Color Scheme: Three complementary colors (Green #4CAF50, Cream #F9F7F3, Brown #5D4037).
